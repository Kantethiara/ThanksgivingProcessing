# Analyse Exploratoire et Visualisation de Données sur la Fête de Thanksgiving

Ce projet utilise Python, NumPy, Pandas, et Matplotlib pour réaliser une analyse exploratoire des données (EDA) et visualiser des informations tirées d'une base de données sur la fête de Thanksgiving.

## Introduction

Ce projet se concentre sur l'analyse des données relatives à la fête de Thanksgiving. L'objectif est de comprendre les tendances, les comportements, et les préférences des gens lors de cette fête grâce à l'analyse de données et à la visualisation.

## Technologies Utilisées

- Python 3
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook (facultatif pour exécuter le code dans un environnement interactif)

## Installation

1. Clonez ce dépôt sur votre machine locale :

    ```bash
    git clone https://github.com/Kantethiara/ThanksgivingProcessing
    ```

2. Accédez au dossier du projet :

    ```bash
    cd ThanksgivingProcessing
    ```

## Utilisation

1. Téléchargez la base de données sur la fête de Thanksgiving (si elle n'est pas incluse dans le dépôt).

2. Lancez le script d'analyse et de visualisation :

    ```bash
    python THIARA_KANTEYE_NotebookProjetFinalPython_MSI2022_2023.ipynb
    ```

3. Pour visualiser les graphiques, vous pouvez utiliser un notebook Jupyter en ouvrant le fichier `.ipynb` fourni :

    ```bash
     THIARA_KANTEYE_NotebookProjetFinalPython_MSI2022_2023.ipynb
    ```

## Résultats

Les résultats de l'analyse incluent des visualisations qui montrent les préférences alimentaires, les réponses démographiques et les traditions associées à la fête de Thanksgiving. Ces informations peuvent être utiles pour comprendre les habitudes des consommateurs pendant cette période de l'année.
